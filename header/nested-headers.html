Problem: Nesting header elements is inappropriate and confuses assistive technologies about the document structure. Header elements should not be nested inside each other as it creates an unclear hierarchy that can be difficult for screen reader users to navigate.

Correct Implementation

Main Header

Sub Header

Solution: If you need to include subheadings within a header element, use appropriate heading levels (h1-h6) wrapped in regular divs or other appropriate elements instead of nesting header elements. This maintains a clear document structure while still allowing for visual organization of headings.

Alternatively, you can use a sectioning element to properly separate content:


<header>
    <h2>Main Header</h2>
</header>
<section>
    <header>
        <h3>Sub Section Header</h3>
    </header>
    <p>Section content...</p>
</section>

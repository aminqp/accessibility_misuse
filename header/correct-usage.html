Why this is good:

  • The page includes a skip link that allows keyboard users to bypass the header
  • The page header contains a clear h1 title identifying the site
  • The header includes only appropriate header content (logo, title, main navigation)
  • The navigation within the header has a proper aria-label
  • The article has its own semantic header for article-specific introductory content
  • Each header has at least one heading (h1-h6) element
  • The document follows a logical structure with clear separation between header, main content, and footer

Why section-level headers are good:

  • Each section has its own header with appropriate heading levels
  • Headers contain descriptive introductory content for their sections
  • The heading hierarchy is maintained (h2 for the main section, h3 for subsections)
  • Each header clearly relates to its section and provides context
  • The headers help screen reader users understand the structure of the content

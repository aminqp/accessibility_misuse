This demonstrates proper usage where:

  • Each nav element has a descriptive aria-label to identify its purpose
  • Nav elements contain actual navigation links
  • Each nav has a heading (visually or via aria-labelledby)
  • Nav elements are used only for significant navigation sections
  • Different types of navigation (primary, secondary, footer) are properly labeled

Best practices demonstrated:

  • Primary navigation in the header is clearly labeled
  • Specialized navigation (Blog categories) in the sidebar has its own descriptive label
  • Footer navigation has a distinct label to distinguish it from other navigation elements
  • Each navigation section uses proper list markup for the links
  • Navigation headings provide additional context for all users

Correct Usage Example: Sectional Footer

Why sectional footers are good:

  • The footer is properly placed within the article element it relates to
  • It contains metadata about the article (author, publication date, categories)
  • The footer uses the time element with a machine-readable datetime attribute
  • Since it's within an article, it doesn't get the contentinfo landmark role, avoiding confusion with the main page footer
  • The content is clearly related to the article it appears in, maintaining proper semantic relationships

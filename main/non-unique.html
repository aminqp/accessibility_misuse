Footer content

Solution: Place navigation elements and other recurring content outside the main element. The main element should only contain the primary content unique to the current page. Navigation menus should be placed in nav elements either in the header, aside, or footer depending on their purpose and importance.

Important Notes
  • The <main> element should contain content that is unique to the document
  • Navigation, site branding, search forms, and other content repeated across the site should be placed outside the main element
  • If site navigation must appear in different places throughout the site, it should be placed within <nav> elements in the appropriate landmarks (header, footer, aside), not in the main element
  • When users navigate by landmarks, they expect the main element to contain the primary content of the page, not auxiliary content like navigation
  • Use appropriate landmark elements for different types of content to create a clear document structure:
    • <header> for introductory content and primary navigation
    • <nav> for navigation blocks
    • <main> for the primary unique content
    • <aside> for tangentially related content
    • <footer> for site footer information

Why the Main Element Is Essential

The <main> element serves several important functions that are crucial for accessibility:

Landmark Navigation

Screen reader users often navigate pages by landmarks. The main landmark is a critical target that allows them to jump directly to the primary content, skipping navigation and other repetitive elements.

Screen Reader Shortcuts

Many screen readers provide keyboard shortcuts to navigate directly to the main content:

  • NVDA: Insert + M
  • JAWS: R (in landmark navigation mode)
  • VoiceOver: Control + Option + M (on macOS)

Semantic Structure

The <main> element creates a clear semantic structure for the document. This helps both users and automated tools understand the organization of the content.

Skip Links

The <main> element provides a logical target for skip links, which are especially important for keyboard users who would otherwise need to tab through all navigation items before reaching the content.

Automated Testing

Many accessibility testing tools check for the presence of a main element as part of their automated tests. Missing this element can result in failing accessibility checks.

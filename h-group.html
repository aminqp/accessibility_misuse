Issue 2: Using headings for styling

Using heading tags just to make text look a certain way is problematic. Headings should represent the document structure.

Accessibility Best Practices for Headings

  • Use headings to create a logical document structure
  • Don't skip heading levels (h1 → h2 → h3, not h1 → h3)
  • Use only one h1 per page
  • Don't use headings purely for styling
  • Ensure all headings have meaningful content
  • Use headings to represent actual sections, not for emphasis
